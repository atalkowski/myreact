Guidewire Senior Software Engineer.

What you bring to Guidewire

  • Strong Programming and software development skills in one or more of the following frameworks & languages: Java / Golang / Python / equivalent
  • Container technologies: Docker / equivalent in a Cloud ecosystem
  • Team first attitude, coupled with curiosity, high potential to learn new technologies and gain strong technical skills
  • Comfort working in an agile, social and fast-paced environment

  • BS/MS degree (Computer Science or Math) or equivalent level of proven professional proficiency
